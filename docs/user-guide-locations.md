# 网点管理操作手册

> BHT 管理端 — 机器/点位全生命周期管理

---

## 1. 查看所有机器

页面顶部工具栏：

- **搜索框**：支持按机器编号（如 A11(2)）、名称（如 Office A20）、区域名搜索
- **区域筛选**：下拉选择区域（Dar es Salaam、Mbagala、TBD 等），默认 "ALL AREAS"
- **状态筛选**：全部状态 / ⚠️ 有异常报告 / 🔧 维护中 / 💔 故障

机器以**卡片网格**展示，每张卡片显示：
- 机器照片（或 "No Photo" 占位图）
- 状态标签（运营中/维护中/停用）
- 机器名称 + 编号
- 上次读数、佣金率
- 启动债务余额 + 还款百分比
- 联系方式、照片/定位状态

---

## 2. 编辑机器信息

点击卡片上的 **Edit site** 按钮，弹出编辑表单。

**可编辑字段：**

| 字段 | 说明 |
|------|------|
| 名称 | 机器名称 |
| 区域 | 所在区域 |
| 机器编号 | 唯一标识 |
| 纬度/经度 | GPS 坐标（-90~90 / -180~180） |
| 佣金率 | 百分比，如 15 |
| 上次读数 | 手动修正 counter |
| 状态 | active / maintenance / broken |
| 机主姓名/电话 | 店主信息 |
| 分配司机 | 该机器的负责司机 |
| 启动债务 | 初始债务/剩余债务 |

**保存后触发：**
- 修改佣金率或债务 → 写入 finance_audit_log 审计
- 修改区域或坐标 → 自动更新 last_relocated_at

## 3. 删除机器

**7 项前置阻塞检查，任一项不通过则无法直接删除：**

| 阻塞条件 | 提示 |
|----------|------|
| remainingStartupDebt > 0 | 未清启动债务 |
| dividendBalance > 0 | 未付业主分红余额 |
| resetLocked = true | 重置锁定状态 |
| 有待处理重置申请 | 审批中的重置 |
| 有待处理提现申请 | 审批中的提现 |
| 有未结算收款 | 未日结的 collection |
| 有等待审批的交易 | approvalStatus=pending |

**删除流程：**
1. 先点 **🔓 强制清除阻塞** — 清零债务/余额/锁定（需确认）
2. 系统自动解绑关联司机
3. 点 **删除此点位** → 确认对话框 → 确认删除

**级联影响：** 历史交易保留但 locationId 置 NULL，关联变更请求级联删除。

## 4. 注册新机器

在司机端 **HARAKA** 页面点 **Register New Machine**，或在管理端机器注册表单。

**必填字段：**
- 机器编号（唯一，不可重复）
- 名称
- GPS 坐标（纬度/经度）

**选填字段：**
- 区域、机主姓名/电话、分配司机
- 启动债务
- 机器照片

**验证规则：**
- 机器编号非空且唯一
- 纬度 -90~90，经度 -180~180
- 所有带 * 的必填项

保存后立即出现在机器列表中。

---

## 5. 照片管理

每张机器卡片顶部显示照片区：
- 有照片 → 显示缩略图 + "查看" 按钮放大查看
- 无照片 → "No Photo" 占位

**上传照片：** 编辑机器时通过文件选择器上传（jpg/png，≤5MB）
**存储位置：** Supabase Storage，通过 `getOptimizedImageUrl()` 加载优化版本

---

## 6. 卡片界面说明

每张机器卡片底部信息栏：

| 图标 | 含义 |
|------|------|
| 📞 | 机主电话（无则显示"无"） |
| 📷 | 照片状态（已上传/无） |
| 📍 | GPS 定位（已定位/无） |
| 🆕 | 新点位标记 |

**启动债务进度条：** 绿色百分比 = 已还比例，旁边显示剩余金额。

---

⚠️ 本手册对应 BHT v1.0.15，如有更新请查最新文档。
